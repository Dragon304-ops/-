# Shosho Crochet

![Shosho Crochet Logo](logo.png)

مرحبًا بكم في **Shosho Crochet**، موقع إلكتروني لعرض منتجات كروشيه.  

## الميزات | Features
- عرض المنتجات بشكل أنيق مشابه لموقع أمازون | Display products in an Amazon-like layout
- سلة مشتريات تفاعلية | Interactive shopping cart
- خانة عنوان التوصيل | Delivery address input
- رقم هاتف للتواصل | Contact phone number
- زر واتساب مباشر للتواصل السريع | Direct WhatsApp button

## المنتجات | Products
- Shosho Crochet Scrunchie  
- Crochet Keychain  
- Handmade Crochet Bag
